# Como Zerar a Fatura do Cartão de Crédito

Quando a fatura consome todo o salário, você vira refém do cartão: paga,
zera a conta, e precisa voltar a passar tudo no cartão até o próximo mês.
O ciclo só quebra com um plano de redução gradual.

## Por que você caiu no ciclo

Bancos e bandeiras usam iscas (milhas, cashback, sala VIP) para concentrar
todo seu poder de compra no cartão. Cada compra rende taxa para o lojista,
antecipação para a operadora e, quando você atrasa, juros para o banco.

O resultado típico:

```
Salário R$ 5.000
- Despesas fixas: R$ 3.000
- Fatura do cartão: R$ 2.000
= R$ 0 livres → vida toda no cartão de novo
```

## Passo 1 — Separe despesas em fixas e diárias

### Saídas fixas (pouca gerência no curto prazo)

Financiamento, aluguel, plano de saúde, condomínio, escola, energia, água,
gás. Variam pouco e mudar exige esforço grande.

### Diárias (alta gerência)

Mercado, lanches, restaurante, transporte, compras por impulso. Variam
conforme humor, fome, vontade.

**Diagnóstico**: se as fixas já consomem quase tudo, o problema é estrutural
e exige decisões grandes. Se as fixas deixam folga, o problema está no
diário (que está indo todo para a fatura).

## Passo 2 — Calcule seu teto diário

```
Salário             R$ 5.000
- Despesas fixas    R$ 3.000
= Disponível p/ dia R$ 2.000
÷ 30 dias           ~R$ 67/dia
```

Esse R$ 67 é o que você pode gastar por dia em despesas diárias **somando
cartão e débito**.

## Passo 3 — Inverta o caminho

Você entrou no ciclo gastando tudo no cartão. Sai dele transferindo gasto
para o débito.

1. **Reduza o gasto no cartão para abaixo do teto diário**. Ex.: gaste
   R$ 33 por dia no cartão. A próxima fatura vem ~R$ 1.000 (metade).
2. **Use o que sobrou no débito** no mês seguinte. Não reabasteça o cartão.
3. **A fatura cai mais** no mês seguinte → mais dinheiro livre no débito.
4. Repita até a fatura caber confortavelmente no orçamento.

## Visão do ciclo virtuoso

```
Fatura cai → sobra dinheiro livre → débito cobre mais despesas
        ↑                                       ↓
        └── fatura cai ainda mais no mês seguinte
```

## Reforços que aceleram o processo

- **Reduzir despesas fixas** quando possível (cancelar serviços não
  essenciais, renegociar planos).
- **Aumentar a renda**: a diferença vai para o débito, nunca para inflar
  a fatura do cartão.

## Princípio

A liberdade do cartão exige **dinheiro livre fora dele**. Sem sobra, não há
débito. Sem débito, a fatura do mês seguinte volta cheia. Reduzir gasto no
cartão é o único atalho real.
